# DVD Rental Customer Segmentation

SQL-based customer segmentation analysis on the PostgreSQL dvdrental sample database. Groups customers by rental behavior using RFM (Recency, Frequency, Monetary) analysis.

## Approach

- Connected to PostgreSQL dvdrental database
- Wrote SQL queries to calculate RFM metrics per customer
- Segmented customers into groups based on rental patterns
- Identified high-value customers and churn risks

## Tech Stack

- PostgreSQL, SQL
- Data modeling and aggregation queries

## Files

- `customer_segmentation.sql` — Full SQL analysis with RFM scoring
- `schema.md` — Database schema reference
